## 转义字符

在 Markdown 中，您可以使用反斜杠（\）来转义特殊字符，以防其具有特殊含义。例如，要显示一个星号（*），您可以使用 `\*`。

\* 这是一个星号，而不是一个列表项。

---

## Markdown 中的 HTML

Markdown 支持使用 HTML 标签来格式化文本。例如，您可以使用 `<em>` 标签来创建斜体文本：

<em>斜体文本</em>

但是，Markdown 语法具有更高的优先级，因此在使用 HTML 标签时，应避免使用与 Markdown 语法相同的标签。例如，如果您想创建一个链接，请使用 Markdown 的链接语法，而不是使用 HTML 的 `<a>` 标签。
